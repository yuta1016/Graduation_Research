This is the README file for the MPEG-7 Reference Software.
It includes links where to find other useful documentation about
the reference software. To get information about

- the philosophy of this software see the MPEG- Documents:
  * ISO/IEC 15938-6 Information Technology - Multimedia Content Description
    Interface - part 6: Reference Software
  * MPEG-7 Overview
- how to compile the XM software see the MPEG-7-build-run-instruction.docx file
  coming with this archive in the src directory.
- the function of each Descriptor or DescriptionScheme see the .txt files
  in the directories Doc/Audio, Doc/MDS, and Doc/Video of this source tree
