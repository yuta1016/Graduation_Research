The International Standard, also known as "Multimedia Content Description Interface," provides a standardized set of technologies for describing multimedia content. It addresses a broad spectrum of multimedia applications and requirements by providing a metadata system for describing the features of multimedia content.

More information on MPEG-7 XM Software can be found in: ISO/IEC 15938-6 Information Technology - Multimedia Content Description Interface - part 6: Reference Software.

This documentation provides usage information for individual Descriptors and Description Schemes.
