Summary The image signature (identifier) is a descriptor that identifies an image. The descriptor is robust to image modifications such as rotation, compression and greyscale conversion. Searching is performed using the Hamming distance, which makes it very fast.
Strong Points Compact, robust and highly independent. Very fast searching
